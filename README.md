Welcome to my **Platform Engineering Learning Repo**! This project serves as a centralized hub where I document my entire journey into cloud-native technologies, compiling everything from automation scripts and infrastructure-as-code configurations to architectural diagrams and study notes.

Whether you are a beginner looking for tutorials or an experienced engineer looking for quick reference scripts, this repository is designed to help you learn and build. Feel free to explore the directories, try out the hands-on labs, and even contribute your own findings to help the community grow.

| Directory | What's Inside |
| :--- | :--- |
| `scripts/` | Bash and Python automation tools for infrastructure tasks |
| `diagrams/` | Visual architecture maps (Mermaid, PNG, SVG) |
| `configs/` | Sample manifests for Terraform, Kubernetes, and Helm |
| `docs/` | Tutorials, concept explanations, and cheat sheets |
